# 商品售卖小程序

一个简单的微信小程序,支持商品上传和售卖功能。

## 功能特点

### 1. 商品上传
- 选择/拍摄商品图片
- 填写商品名称、价格、库存
- 添加商品描述
- 查看已上传商品列表
- 删除商品

### 2. 商品展示
- 首页展示热门商品
- 商品列表页(网格布局)
- 商品详情页

### 3. 购物车
- 加入购物车
- 调整商品数量
- 删除商品
- 查看总价

## 技术实现

- **数据存储**: 使用微信小程序本地存储(wx.setStorageSync)
- **状态管理**: App 全局数据管理
- **UI设计**: 简洁现代的卡片式布局

## 使用方法

### 1. 导入项目
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `shop-miniapp` 文件夹
4. 填写 AppID(测试可用测试号)

### 2. 上传商品
1. 点击底部"上传"标签
2. 选择商品图片
3. 填写商品信息
4. 点击"发布商品"

### 3. 浏览购买
1. 在首页或商品页浏览商品
2. 点击商品查看详情
3. 加入购物车
4. 在购物车页面结算

## 目录结构

```
shop-miniapp/
├── app.js              # 小程序入口
├── app.json            # 全局配置
├── app.wxss            # 全局样式
├── pages/
│   ├── index/          # 首页
│   ├── product/
│   │   ├── list/       # 商品列表
│   │   └── detail/     # 商品详情

│   └── admin/
│       └── upload/     # 商品上传
└── sitemap.json        # 索引配置
```

## 注意事项

1. **图片存储**: 当前使用本地临时路径,重启小程序后图片会失效。生产环境需要上传到服务器。

2. **数据持久化**: 使用本地存储,数据仅保存在当前设备。多设备同步需要后端支持。

3. **支付功能**: 当前未实现真实支付,需要接入微信支付 API。
